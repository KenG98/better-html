Making a beautiful, responsive, static website should be easy, fast, and fun.

What is BetterHTML?
BetterHTML is a super easy way to create responsive, static websites which look great on mobile and desktop. BetterHTML code is easier to write, easier to understand, and shorter.

This website is written entirely in BetterHTML - explore it on your computer and your phone to see how it handles screen sizes, then install BetterHTML, read the tutorial, and create a great website of your own.

Why BetterHTML?
Looks Great
This website is written in BetterHTML - resize the window to see how it is mobile friendly and responsive.
Less Code
Comparing BetterHTML to traditional HTML, the code for this page has 35% fewer lines and 50% fewer characters.
Easy to Use
Check out the code comparison below - BetterHTML is easier to understand and write than regular HTML.
BetterHTML
- row (align center):
  - Column 1 text goes here!
  - Column 2 goes here.
  - Finally, column 3.
Regular HTML
<div class="pure-g" style="text-align:center;">
  <div class="pure-u-1 pure-u-md-1-3">
    <p>Column 1 text goes here!</p>
  </div>
  <div class="pure-u-1 pure-u-md-1-3">
    <p>Column 2 goes here.</p>
  </div>
  <div class="pure-u-1 pure-u-md-1-3">
    <p>Finally, column 3.</p>
  </div>
</div>

Getting Started
Install BetterHTML
BetterHTML is an NPM package. First install Node.js at nodejs.org. Then run `npm install betterhtml --global`.
Visit The Tutorial
Read the official BetterHTML tutorial at better-html.com/tutorial/1-install.html.
